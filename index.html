系統說明與 Micro:bit USB 設定

如何使用：

  1. 將您的 Micro:bit 透過 USB 傳輸線 連接至電腦。
  2. 點擊本網頁上方的「連接 Micro:bit (USB)」按鈕，選擇對應的 COM 埠。
  3. 連接成功後，Micro:bit 就會不斷向網頁傳送數據。
  4. 點選圖表標題旁邊的 「▶️ 開始測量」 按鈕，系統就會將接收到的數據導入該圖表與動畫中。
  5. 測試完一般建築物後，點選停止，再換點選隔震結構的「開始測量」繼續測試。

Micro:bit MakeCode 極簡版程式寫法：

現在網頁具有超強容錯力，Micro:bit 只需要單純地寫入強度數值 即可，網頁上的按鈕會負責決定資料要送到哪裡！

// 1. 在「當啟動時」設定 USB 序列埠
serial.redirectToUSB()

// 2. 在「重複無限次」中，直接把加速度數值寫成一行傳出去
basic.forever(function () {
    let accel = input.acceleration(Dimension.Strength)
    
    // 只要這一行！網頁會自動過濾並接收數字
    serial.writeLine("" + accel)
    
    basic.pause(200)
})

※ 注意：Web Serial API 必須使用 Google Chrome 或 Edge 瀏覽器。若在預覽視窗無法彈出連接畫面，請「下載」此 HTML 檔案並放在電腦上，直接雙擊開啟即可。
