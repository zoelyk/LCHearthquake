系統說明與 Micro:bit 設定

如何使用：

  1. 將您的 Micro:bit 燒錄相對應的程式。
  2. 點擊「連接 Micro:bit」按鈕。
  3. 在彈出的視窗中選擇您的 Micro:bit 裝置進行配對。
  4. 連接成功後，開啟您的震動平台，圖表將每 0.2 秒即時更新加速度數據，下方動畫也會根據真實強度運算傢俬傾倒的物理反應。

Micro:bit MakeCode 設定提示：

  • 需加入 Bluetooth 擴充功能。
  • 在「專案設定」中，必須開啟 「No Pairing Required: Anyone can connect via Bluetooth」。
  • 使用藍牙 UART 服務傳送數據，格式請設定為：一般建築數值,隔震結構數值 （例如寫入字串 1024,800）。

※ 注意：若要在預覽視窗連接藍牙，需確保瀏覽器支援 Web Bluetooth API，且由於安全性限制，建議下載 HTML 檔案後在本地端直接開啟網頁執行。
