系統說明與 Micro:bit USB 設定

如何使用：

  1. 將您的 Micro:bit 透過 USB 傳輸線 連接至電腦。
  2. 點擊本網頁上方的「連接 Micro:bit (USB)」按鈕。
  3. 在彈出的視窗中，選擇您的 Micro:bit 所在的 COM 埠（通常名稱會包含 mbed Serial Port 或 BBC micro:bit CMSIS-DAP），然後點選「連線」。
  4. 連接成功後，開啟您的震動平台，即可即時接收數據。

Micro:bit MakeCode 程式寫法參考：

請使用 序列埠 (Serial) 相關積木，不要用藍牙積木。Baud Rate 預設使用 115200。

let testMode = 0

// 初始化時可將序列埠導向 USB，Baud Rate 設定 115200
serial.redirectToUSB()

input.onButtonPressed(Button.A, function () {
    testMode = 0
    basic.showString("L")
})

input.onButtonPressed(Button.B, function () {
    testMode = 1
    basic.showString("R")
})

basic.forever(function () {
    let accel = input.acceleration(Dimension.Strength)
    
    // 透過 USB 序列埠傳送數據，並記得加上換行符號
    if (testMode == 0) {
        serial.writeString(accel + ",1000\n")
    } else {
        serial.writeString("1000," + accel + "\n")
    }
    basic.pause(200)
})

※ 注意：Web Serial API 必須使用 Google Chrome 或 Edge 瀏覽器。若在預覽視窗無法彈出連接畫面，請「下載」此 HTML 檔案並放在電腦上，建議透過本機伺服器 (Localhost) 或直接雙擊開啟來執行。
